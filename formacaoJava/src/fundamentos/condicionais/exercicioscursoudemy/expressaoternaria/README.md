# Exercício: Expressão Ternária

## Descrição

Programa que lê um número inteiro e informa se ele é par ou ímpar utilizando o operador ternário.

## Conceitos Praticados

- Operador ternário (`? :`)
- Estruturas condicionais
- Operador módulo (`%`)
- Entrada de dados com `Scanner`

## Regra de Negócio

- Se o número for divisível por 2, o programa exibe `PAR`.
- Caso contrário, exibe `IMPAR`.

## Exemplo

```text
Digite um valor: 10

PAR
```
