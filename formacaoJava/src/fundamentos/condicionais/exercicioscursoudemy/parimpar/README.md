# Exercício: Par ou Ímpar

## Descrição

Programa com menu interativo que permite ao usuário verificar se um número é par ou ímpar.

O sistema permanece em execução até que o usuário escolha a opção de finalizar.

## Conceitos Praticados

- Estruturas condicionais (`if/else`)
- Estrutura de repetição (`while`)
- Estrutura de seleção (`switch`)
- Operador módulo (`%`)
- Entrada de dados com `Scanner`

## Funcionalidades

1. Testar se um número é par ou ímpar.
2. Encerrar o programa.

## Exemplo

```text
--- MENU ---

1 - Testar se um número é par
2 - Finalizar programa

Escolha uma opção: 1

Digite um número: 15

O número é ímpar
```
