# Exercício: Múltiplos

## Descrição

Programa com menu interativo que permite verificar se dois números são múltiplos entre si.

O sistema permanece em execução até que o usuário escolha a opção de finalizar.

## Conceitos Praticados

- Estruturas condicionais (`if/else`)
- Estrutura de repetição (`while`)
- Estrutura de seleção (`switch`)
- Operador módulo (`%`)
- Entrada de dados com `Scanner`

## Regra de Negócio

Dois números são considerados múltiplos quando:

```java
numero1 % numero2 == 0
```

ou

```java
numero2 % numero1 == 0
```

## Funcionalidades

1. Verificar se dois números são múltiplos.
2. Encerrar o programa.

## Exemplo

```text
Digite o primeiro número: 10
Digite o segundo número: 5

São múltiplos!
```
