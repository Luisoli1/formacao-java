# Problema: Maior Posição

## Descrição

Programa que lê `N` números reais, armazena os valores em um vetor e identifica o maior valor digitado.

Ao final, exibe o maior valor e a posição em que ele aparece no vetor, considerando que a primeira posição é `0`.

## Conceitos praticados

- Arrays
- Laço `for`
- Comparação de valores
- Controle de maior valor
- Índices de array

## Exemplo de saída

```text
MAIOR VALOR = 14.0
POSICAO DO MAIOR VALOR = 3
```
