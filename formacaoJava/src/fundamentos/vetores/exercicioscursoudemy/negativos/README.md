# Problema: Negativos

## Descrição

Programa que lê um número inteiro positivo `N`, armazena `N` números inteiros em um vetor e exibe na tela apenas os números negativos digitados.

## Conceitos praticados

- Arrays
- Laço `for`
- Condicionais
- Leitura de dados com `Scanner`
- Verificação de números negativos

## Exemplo de saída

```text
NUMEROS NEGATIVOS:
-2
-3
-7
```
