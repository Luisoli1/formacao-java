# Problema: Soma Vetor

## Descrição

Programa que lê `N` números reais, armazena os valores em um vetor, imprime todos os elementos, calcula a soma e exibe a média dos valores.

## Conceitos praticados

- Arrays
- Laço `for`
- Acumulador
- Média aritmética
- Leitura de números reais

## Exemplo de saída

```text
VALORES = 8.0 4.0 10.0 14.0
SOMA = 36.00
MEDIA = 9.00
```
