# Problema: Pensionato

## Descrição

Programa que simula o controle de aluguel de quartos de um pensionato.

O pensionato possui 10 quartos, identificados pelos números de 0 a 9. O programa lê a quantidade de estudantes que irão alugar quartos e, para cada aluguel, registra o nome, o e-mail e o número do quarto escolhido.

Ao final, é exibido um relatório com todos os quartos ocupados, em ordem crescente pelo número do quarto.

---

## Regras de Negócio

1. O pensionato possui 10 quartos.
2. Os quartos são identificados pelos números de 0 a 9.
3. O usuário informa a quantidade de estudantes que irão alugar quartos.
4. Para cada estudante, devem ser informados:
   - Nome
   - E-mail
   - Número do quarto
5. Cada quarto ocupado armazena os dados de um aluguel.
6. Ao final, o programa exibe apenas os quartos ocupados.
7. O relatório final é exibido em ordem crescente pelo número do quarto.
