# Problema: Alturas

## Descrição

Programa que lê nome, idade e altura de `N` pessoas, armazenando cada informação em arrays separados.

Ao final, exibe a altura média, a porcentagem de pessoas com menos de 16 anos e os nomes dessas pessoas.

## Observação

Neste exercício, os dados foram armazenados em arrays separados, conforme orientação do curso, sem criação de classes.

## Conceitos praticados

- Arrays paralelos
- Laço `for`
- Cálculo de média
- Cálculo de porcentagem
- Condicionais
- Entrada de dados com `Scanner`

## Exemplo de saída

```text
Altura média: 1.69
Pessoas com menos de 16 anos: 40.0%
Joao
Teresa
```
