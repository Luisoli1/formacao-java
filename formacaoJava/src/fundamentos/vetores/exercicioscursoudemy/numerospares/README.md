# Problema: Números Pares

## Descrição

Programa que lê `N` números inteiros, armazena os valores em um vetor e exibe todos os números pares digitados.

Ao final, também mostra a quantidade total de números pares encontrados.

## Conceitos praticados

- Arrays
- Laço `for`
- Operador módulo `%`
- Condicionais
- Contador

## Exemplo de saída

```text
NUMEROS PARES:
8 2 14 20

QUANTIDADE DE PARES = 4
```
