# Estudos de Manipulação de Strings em Java

Este repositório documenta meus estudos iniciais sobre a classe `String` em Java. Durante a minha trilha de aprendizado, criei este exercício para praticar a manipulação de textos e dados, aplicando os métodos essenciais que venho estudando em aula.

## Sobre o Exercício
O objetivo deste código é aplicar de forma prática os métodos de manipulação de `String` (como `trim`, `substring`, `replace`, etc.) em um cenário que simula o tratamento de dados de entrada. Utilizei uma estrutura que remete a um sistema de gestão para tornar o aprendizado mais próximo de problemas reais de desenvolvimento.

## Métodos Praticados
Neste código, apliquei:
- **Formatação**: `trim()`, `toUpperCase()`, `toLowerCase()`
- **Recorte**: `substring()`
- **Substituição**: `replace()`
- **Busca**: `indexOf()`
- **Divisão**: `split()`

## Objetivo
O foco principal deste repositório é demonstrar a compreensão das funções básicas da linguagem Java e o cuidado com a manipulação de dados — competências que estou desenvolvendo ativamente na minha transição de carreira para a área de tecnologia.
