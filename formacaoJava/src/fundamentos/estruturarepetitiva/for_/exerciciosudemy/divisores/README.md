# Exercício: Divisores

## Descrição

Programa que lê um número inteiro e exibe todos os seus divisores.

## Conceitos Praticados

- Estrutura de repetição `for`
- Operador módulo `%`
- Estruturas condicionais
- Entrada de dados com `Scanner`

## Regra de Negócio

- Percorrer os números de 1 até N.
- Verificar quais valores dividem N sem deixar resto.
- Exibir todos os divisores encontrados.

## Exemplo

```text
Entrada:
6

Saída:
1
2
3
6
```
