# Exercício: Números Ímpares

## Descrição

Programa que lê um número inteiro e exibe todos os números ímpares de 1 até o valor informado.

## Conceitos Praticados

- Estrutura de repetição `for`
- Operador módulo `%`
- Estruturas condicionais
- Entrada de dados com `Scanner`

## Regra de Negócio

- Percorrer os números de 1 até N.
- Verificar quais são ímpares.
- Exibir cada número ímpar encontrado.

## Exemplo

```text
Entrada:
8

Saída:
1
3
5
7
```
