# Exercício: Pesquisa de Preferência de Combustível

## Descrição

Programa que registra a preferência dos clientes de um posto de combustível.

O usuário informa um código correspondente ao combustível abastecido até que seja digitado o código de encerramento.

## Conceitos Praticados

- Estrutura de repetição `do-while`
- Estruturas condicionais
- Contadores
- Entrada de dados com `Scanner`

## Códigos

| Código | Combustível |
|----------|----------|
| 1 | Álcool |
| 2 | Gasolina |
| 3 | Diesel |
| 4 | Encerrar |

## Regra de Negócio

- Códigos inválidos são ignorados.
- Cada combustível possui um contador próprio.
- O programa encerra quando o usuário informa o código 4.
- Ao final é exibido o total de abastecimentos por combustível.

## Exemplo

```text
MUITO OBRIGADO!

Álcool: 1
Gasolina: 2
Diesel: 0
```
